## 任务
- 模仿B站百万粉丝博主笔风，以讲故事形式创作约1500字爆款文案（开头100字 + 赏析1000字 + 结尾100字）
- 主题：陈子昂《登幽州台歌》

## 爆款公式
```
爆款文案 = 钩子开头 + 故事化叙事 + 情绪共鸣 + 升华结尾
```

## 创作流程

### Step 1：生成5个开头示例（黄金5秒法则）
- 核心逻辑：交叉验证标签式开头
  - 标签A（纵向）：历史地位背书（如“初唐文坛第一炸场者”）
  - 标签B（横向）：独特身份定位（如“被官场耽误的孤独诗人”）
  - 要求：A与B形成反差张力，避免同维度描述
- 句式选择（5种适配主题的句式）：
  1. **对撞式**：“他是初唐诗坛的孤勇者，也是官场失意的弃子。”
  2. **反差式**：“世人皆知他的诗名，却不知他的绝望。”
  3. **设问式**：“是什么让一代才子发出‘前不见古人’的悲鸣？”
  4. **场景还原式**：“公元696年，幽州台上，陈子昂拔剑问天。”
  5. **直接断言式**：“这是中国文学史上最孤独的17个字。”
- 输出格式：列出5个完整开头，标注句式类型

### Step 2：生成5个结尾示例（升华留余味）
- 核心公式：总结回顾 + 金句升华 + 留白余味
- 结尾类型（差异化呈现）：
  1. **凝练式**：“千年后，我们仍能从这17个字中，读懂人类共通的孤独。”
  2. **意象式**：“幽州台的风还在吹，吹过每个不甘平庸的灵魂。”
  3. **展望式**：“当你站在人生的幽州台，别忘了陈子昂的呐喊：天地虽大，总有属于你的位置。”
- 输出格式：列出5个完整结尾，标注类型

### Step 3：用户确认后写作全文
- 赏析部分要求：
  1. 按诗句顺序展开，融入历史背景（如武则天时期的政治环境、陈子昂的仕途坎坷）
  2. 故事化叙事：用场景替代科普（如“他看着台下的武攸宜，手中的奏折被汗水浸湿”）
  3. 情绪共鸣：将“前不见古人”的孤独转化为当代人可感的“职场失意”“理想破灭”等情绪
  4. 节奏控制：每段300字左右设置一个情绪转折（从壮志凌云到绝望呐喊）

## 写作禁忌
- ❌ 禁用百科式罗列（如“陈子昂，字伯玉，梓州射洪人”）
- ❌ 禁用说教式表达（如“我们要学习他的爱国精神”）
- ❌ 禁用元叙事引导语（如“让我们分析一下”）
- ❌ 禁用现代网络流行语（如“YYDS”“内卷”）
- ❌ 禁用第二人称对话（如“你知道吗”）
